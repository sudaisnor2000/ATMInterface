# ATM Interface

Project 3 for DecodeLabs Java Internship.

## Features

* Deposit Money
* Withdraw Money
* Check Balance
* Exit ATM
* Input Validation
* Object-Oriented Programming (OOP)

## Classes Used

* BankAccount.java
* ATM.java
* Main.java

## Technologies

* Java
* IntelliJ IDEA
* JDK 25

## How It Works

1. Create a bank account with an initial balance.
2. Deposit money through the ATM menu.
3. Withdraw money if sufficient balance exists.
4. Check current account balance.
5. Exit the application.
